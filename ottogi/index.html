소개

통계학을 전공하고 데이터사이언스대학원에서 공부하며, 데이터를 근거로 원인을 찾는 훈련을 해왔습니다. 노인 노쇠 코호트 데이터를 분석한 대학원 연구에서는 신체·인지·심리처럼 성격이 다른 변수를 정리하면서, 결측값과 이상치를 먼저 꼼꼼히 들여다보지 않으면 분석 결과가 쉽게 틀어진다는 것을 체감했습니다. Boruta로 유의미한 변수를 선별하고 K-means로 군집을 나눠 변수 조합별 위험 양상을 비교하는 과정에서, 데이터 하나하나의 신뢰성부터 확인하는 습관을 길렀습니다.

국가승인통계 공모전에서는 온실가스·발전량·에너지 소비처럼 성격이 다른 여러 공공 데이터를 교차 분석해 정책 결론을 도출한 경험이 있습니다. 하나의 데이터만 보고 판단하지 않고, 여러 출처를 겹쳐봐야 진짜 원인인지 확인할 수 있다는 것을 이때 배웠습니다.

같은 습관을 개인 AI 프로젝트에도 적용하고 있습니다. 데이터 수집부터 검증까지 전 과정을 직접 다루면서, 결과가 좋아 보여도 그대로 믿지 않고 별도 데이터로 다시 확인하는 방식을 일관되게 유지해왔습니다.

학술 발표

그룹별 군집 조합 분석을 통한 노쇠 예측 요인 분석

변수 선택 · 이상치·결측치 처리 · 군집화 · 대학원 개인 연구

2024 대한산업공학회 춘계공동학술대회에서 발표한 개인 연구입니다. 지도교수 지도 하에 데이터 전처리부터 분석·시각화까지 단독으로 수행했습니다. 노인 노쇠 코호트 데이터의 신체·인지·심리 변수군을 정리하며 결측값과 이상치를 먼저 확인하고, Boruta로 노쇠 예측에 유의미한 변수를 선택한 뒤 K-means로 변수군별 군집을 구성해 군집 조합별 위험 양상을 비교했습니다.

  • 데이터 정제결측치·이상치 확인 후 범주형 변수 인코딩
  • 분석 방법Boruta 변수 선택 + K-means 군집화 + SMOTE 불균형 보완
